My Git Cheat Sheet

Git Command

  1. git init (directory)
  2. git status
  3. git diff
  4. git log
  5. git clean -n
  6. git branch

Description

  1. When you want to create an empty git directory
  2. Lists the status of all files
  3. Shows unstaged changes
  4. Shows entire commit history
  5. Shows which files would be removed from working directory
  6. Shows all branches in a repo
